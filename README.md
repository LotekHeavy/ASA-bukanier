<h2>
    Es gibt eine Umzugsoption
</h2>
<p>

Dieser gemeinsame Transfer findet voraussichtlich an einem Wochenende im Juli statt.

<h2>
    Warum wird der Cluster abgeschaltet?
</h2>
<p>
    Es gibt mehrere Gründe. Einerseits haben wir mit <a href="https://www.hetzner.com/de/pressroom/statement-on%20the-latest-adjustment-to%20setup-fees/">erneut steigenden Hosting-Kosten</a> zu kämpfen (ein zukünftig notwendiges <a href="https://www.hetzner.com/de/dedicated-rootserver/ax102/configurator/">Hardware-Upgrade</a> - für weitere Karten - würde monatlich aktuell - kein Scheiss - fast 600 EUR kosten). Andererseits spiele ich selbst schon seit Sommer 2025 aktiv ebenfalls auf "ARK ME HARDER". Wenn dann zusätzlich das Reallife reinkickt, bleiben die Zeit und die Motivation für den Weiterbetrieb eines eigenen Server leider auf der Strecke.
</p>
